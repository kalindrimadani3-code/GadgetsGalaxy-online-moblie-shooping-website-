<h1>Gadgets Galaxy</h1>

<p>Gadgets Galaxy is a full-featured e-commerce website for buying the latest smartphones and gadgets — built with PHP, MySQL, HTML, CSS, and JavaScript.
It includes a powerful User Dashboard and Admin Panel for managing products, orders, and user accounts.</p>
<ul>
  <h1>Features </h1>
  <h2>🛍️ User Side</h2>
<li>🏠 Homepage with featured gadgets and search filters<?li> 
<li>📱 Product Pages for multiple brands (Vivo, Redmi, Samsung, etc.)</li>
<li>🛒 Add to Cart / Wishlist with session-based logic and login check</li>
<li>👤 User Authentication (Login, Register, Logout)</li>
<li>💳 Cart & Checkout with address form and remove-item option</li>
 <li> 🧾 User Dashboard to view:
    Profile details,
    Orders history,
    Wishlist items,
    Update profile or password.</li>
 <h1>🧑‍💼 Admin Side</h1>
  <ul>
  <li>🔐 Secure Admin Login</li>
<li>📦 Product Management:Add / Edit / Delete products Upload images and set prices</li>
<li>📑 Order Management: View all orders placed by users</li>
 <li>👥 User Management: View and manage registered users</li>
<li>📊 Dashboard Analytics: Total users, total orders, sales overview</li>
